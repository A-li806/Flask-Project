# Riz Electrics – Flask Website

## Project Structure
```
riz_electrics/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
└── static/
    └── css/
        └── style.css
```

## Setup & Run

1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the app**
   ```bash
   python app.py
   ```

3. **Open in browser**
   ```
   http://127.0.0.1:5000
   ```

## Features
- Responsive Bootstrap 5 layout
- Sticky navbar with mobile hamburger menu
- Hero section with CTA buttons
- NICEIC credentials bar
- 6 service cards with hover effects
- 24/7 Emergency section
- EICR section for landlords
- Contact form with Flask flash messages
- Social media links in footer
